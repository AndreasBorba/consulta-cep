O projeto

O projeto foi desenvolvido por: Andreas Borba.

O objetivo deste projeto é fornecer informações sobre CEPs utilizando a API da Brasil API.
As tecnologias utilizadas no desenvolvimento incluem Java, Spring Boot, Thymeleaf, HTML, CSS, Bootstrap e JavaScript.
A consulta é realizada com base no CEP fornecido pelo usuário, e os resultados possíveis são:
1. Sucesso: O CEP foi encontrado e as informações correspondentes foram exibidas.
2. Falha: O CEP não foi encontrado.
